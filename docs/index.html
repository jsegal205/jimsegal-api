[GET] /isAnchorageColderThan/:lat/:long

Compares temperatures of the given latitude and longitude to Anchorage, AK, USA
  • :lat - {Float}Latitude of comparision destination
  • :long - {Float}Longitude of comparision destination

Returns

{
    "isAnchorageColder": {Boolean},
    "anchorageDetails": {
        "city": {String},
        "lat": {String},
        "long": {String},
        "state": {String},
        "temperature": {Float},
        "units": {String}
    },
    "compareDetails": {
        "city": {String},
        "lat": {String},
        "long": {String},
        "state": {String},
        "temperature": {Float},
        "units": {String}
    }
}
