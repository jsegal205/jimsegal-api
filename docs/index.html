[POST] /recipes

Endpoint to add more recipes

Required Params

  • :title - {String}title for recipe
  • :ingredients - {String}ingredients for recipe markdown preferred
  • :directions - {String}directions for recipe, markdown preferred

Optional Params

  • :reference_link - {String}link of reference for recipe to another site, url preferred
  • :notes - {String}extra notes pertaining to the recipe

Returns

{
  "title": {String},
  "slug": {String},
  "referenceLink": {String},
  "ingredients": {String},
  "directions": {String},
  "notes": {String}
}
